# Blog
Check it out at [kungfux.github.io](https://kungfux.github.io)

## Commands
- Restore
```bash
$ bundle
```

- Build
```bash
$ bundle exec jekyll
```

- Include drafts into build
```bash
$ bundle exec jekyll --drafts
```

- Build and watch
```bash
$ bundle exec jekyll s
```

- Upgrade theme
```bash
bundle update jekyll-theme-chirpy
```
